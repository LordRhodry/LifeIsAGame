In this game, each turn you chose a valid gravity orientation by pressing the corresponding arrow key. (Valid means it makes tiles move).

If the movement makes two tiles with the same value collide into each other they fuse using the rule of the day.

After Each movement a tile is randomly added to the board.
You lose if there are no valid movement left.

You score points for each fusion you create (the harder the tile you createis to obtain, the more points it gives you).
Good luck!
